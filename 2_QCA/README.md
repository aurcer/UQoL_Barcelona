# UQoL Indicators in Barcelona 2026 - QCA

## Overview
This notebook performs fsQCA on composite UQoL dimension scores for Barcelona neighbourhoods. Tourism status is used as a post-hoc interpretive lens.

## How to run
1. Ensure the PCA notebook has been run first.
2. Place this notebook in the 2_QCA folder.
3. Run all chunks in order.
